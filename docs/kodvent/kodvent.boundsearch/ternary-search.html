ternarySearch

inline fun ternarySearch(left: Double, right: Double, f: (Double) -> Double): Double(source)

Finds the argument that maximizes f on the interval [left, right] using ternary search.

The function f must be unimodal on the given interval, i.e., strictly increasing then strictly decreasing.

Return

the argument x in [left, right] at which f attains its maximum, with precision up to 1e-9.

Parameters

left

the left bound of the search interval.

right

the right bound of the search interval; must be >= left.

f

the unimodal function to maximize.

Throws

IllegalArgumentException

if left>right.

Samples

// Find the peak of a downward-opening parabola f(x) = -(x - 3)^2 + 9
val x = ternarySearch(0.0, 10.0) { x -> -(x - 3) * (x - 3) + 9 }

assertApprox(3.0, x)
