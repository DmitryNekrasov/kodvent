operator fun get(start: Int, end: Int): T

Queries the result of applying the operation over the range [start, end] (inclusive).

This operator function allows using bracket notation for range queries.

Return

the result of applying the operation to all elements in the range [start, end]

Parameters

start

the starting index of the range (inclusive); must be non-negative

end

the ending index of the range (inclusive); must be less than the tree size

Throws

IllegalArgumentException

if the tree is empty or start>end

IndexOutOfBoundsException

if start is negative or end is out of bounds

operator fun get(index: Int): T

Queries the value at a single index position.

This is a convenience operator function that allows using bracket notation with a single index to query individual elements. It is equivalent to calling get(index, index).

Return

the value at the specified index

Parameters

index

the index of the element to query; must be in range [0, size)

Throws

IllegalArgumentException

if the tree is empty

IndexOutOfBoundsException

if index is out of bounds
