fun lcm(a: Int, b: Int): Int

Computes the least common multiple (LCM) of two integers.

The LCM is the smallest positive integer that is divisible by both numbers. This function uses the relationship: lcm(a, b) = (a * b) / gcd(a, b), but calculates it as (a / gcd(a, b)) * b to reduce the risk of integer overflow.

Return

the least common multiple of a and b

Parameters

a

the first integer

b

the second integer
